# Milestone 2.5 — CI + App Platform Deployment Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** Set up GitHub Actions CI (lint + test with real Postgres/Mailpit) and DigitalOcean App Platform deployment (auto-deploy from main).

**Architecture:** Single GitHub Actions workflow with 3 parallel jobs (lint, backend-test, frontend-test). App Platform spec defines a monorepo app with backend, frontend, and managed Postgres components. Release command runs Alembic migrations before each deploy.

**Tech Stack:** GitHub Actions, DigitalOcean App Platform, Postgres 16, Mailpit, uv, bun

### Task 1: GitHub Actions CI Workflow

**Files:**
- Create: `.github/workflows/ci.yml`

**Step 1: Create the workflow file**

```yaml
name: CI

on:
  push:
    branches: [main]
  pull_request:

jobs:
  lint:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4

      - name: Install uv
        uses: astral-sh/setup-uv@v4

      - name: Install bun
        uses: oven-sh/setup-bun@v2

      - name: Backend lint
        working-directory: backend
        run: |
          uv sync --frozen --no-install-project
          uv run ruff check .
          uv run ruff format --check .

      - name: Frontend lint
        working-directory: frontend
        run: |
          bun install --frozen-lockfile
          bun run lint

  backend-test:
    runs-on: ubuntu-latest
    services:
      postgres:
        image: postgres:16
        env:
          POSTGRES_USER: nelson
          POSTGRES_PASSWORD: nelson
          POSTGRES_DB: nelson
        ports:
          - 5432:5432
        options: >-
          --health-cmd "pg_isready -U nelson"
          --health-interval 5s
          --health-timeout 3s
          --health-retries 5
      mailpit:
        image: axllent/mailpit
        ports:
          - 1025:1025
          - 8025:8025
    env:
      DATABASE_URL: postgresql+asyncpg://nelson:nelson@localhost:5432/nelson
      JWT_SECRET: ci-test-secret
      FERNET_KEY: ci-test-fernet-key
      SMTP_HOST: localhost
      SMTP_PORT: 1025
      EMAIL_PROVIDER: smtp
      EMAIL_FROM: nelson@localhost
      CORS_ORIGINS: '["http://localhost:3000"]'
      MAGIC_LINK_BASE_URL: http://localhost:3000/login/verify
    steps:
      - uses: actions/checkout@v4

      - name: Install uv
        uses: astral-sh/setup-uv@v4

      - name: Install dependencies
        working-directory: backend
        run: uv sync --frozen

      - name: Run migrations
        working-directory: backend
        run: uv run alembic upgrade head

      - name: Run tests
        working-directory: backend
        run: uv run pytest -v

  frontend-test:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4

      - name: Install bun
        uses: oven-sh/setup-bun@v2

      - name: Install dependencies
        working-directory: frontend
        run: bun install --frozen-lockfile

      - name: Run tests
        working-directory: frontend
        run: bun run test
```

Expected: `name: CI` visible at top

### Task 2: DigitalOcean App Platform Spec

**Files:**
- Create: `.do/app.yaml`

**Step 1: Create the app spec**

```yaml
name: nelson
region: nyc

alerts:
  - rule: DEPLOYMENT_FAILED
  - rule: DOMAIN_FAILED

services:
  - name: backend
    dockerfile_path: backend/Dockerfile
    source_dir: /
    github:
      repo: 0xc14m1z/nelson
      branch: main
      deploy_on_push: true
    http_port: 8000
    instance_size_slug: basic-xxs
    instance_count: 1
    health_check:
      http_path: /health
      initial_delay_seconds: 15
      period_seconds: 10
    routes:
      - path: /api
        preserve_path_prefix: true
      - path: /health
    envs:
      - key: DATABASE_URL
        scope: RUN_TIME
        value: ${db.DATABASE_URL}
      - key: JWT_SECRET
        scope: RUN_TIME
        type: SECRET
        value: REPLACE_IN_DO_DASHBOARD
      - key: FERNET_KEY
        scope: RUN_TIME
        type: SECRET
        value: REPLACE_IN_DO_DASHBOARD
      - key: EMAIL_PROVIDER
        scope: RUN_TIME
        value: resend
      - key: RESEND_API_KEY
        scope: RUN_TIME
        type: SECRET
        value: REPLACE_IN_DO_DASHBOARD
      - key: EMAIL_FROM
        scope: RUN_TIME
        value: nelson@yourdomain.com
      - key: CORS_ORIGINS
        scope: RUN_TIME
        value: '["https://yourdomain.com"]'
      - key: MAGIC_LINK_BASE_URL
        scope: RUN_TIME
        value: https://yourdomain.com/login/verify

  - name: frontend
    dockerfile_path: frontend/Dockerfile
    source_dir: /
    github:
      repo: 0xc14m1z/nelson
      branch: main
      deploy_on_push: true
    http_port: 3000
    instance_size_slug: basic-xxs
    instance_count: 1
    health_check:
      http_path: /
      initial_delay_seconds: 15
      period_seconds: 10
    routes:
      - path: /
    build_command: ""
    envs:
      - key: BACKEND_URL
        scope: BUILD_TIME
        value: ${backend.PRIVATE_URL}

databases:
  - name: db
    engine: PG
    version: "16"
    size: db-s-1vcpu-1gb
    db_name: nelson
    db_user: nelson
    num_nodes: 1
```
